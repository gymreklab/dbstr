About WebSTR

WebSTR contains population and genome-wide information about genetic variation and other characteristics of short tandem repeats (STRs) in humans. STR call sets are available for multiple cohorts described here which collectively span several thousand individuals and more than one million genomic loci.

WebSTR is based on ExAC (the Exome Aggregation Consortium), but designed for STRs, which are generally not well represented by standard genetic variant callsets not built specifically to deal with repeats.

If you are interested in contributing an STR dataset please contact mgymrek AT ucsd DOT edu. If you would like to report an issue or contribute to WebSTR code, please open an issue on or submit a pull request on the WebSTR Github page.
